## 概要
カセットテープのレイアウトを国際規格（ISO 1001 / IEC 60094-7 / JIS S 6002）に厳密に準拠した配置に修正し、視覚的なスタイルを「AXIA（アクシア）」のカセットテープ（PS-I, GT-II等）にインスパイアされたプレミアムなデザインへと刷新します。

## コンテキスト / 現状
前回の修正でパーツの追加を行いましたが、ユーザーより「設計レイアウトがおかしい」との指摘がありました。目分量での配置ではなく、ミリ単位の設計図に基づいた正確な幾何学配置が求められています。また、デザイン面でもより具体的なブランド（AXIA）を意識したスタイリッシュな外観が期待されています。

## 期待される動作 / ゴール
- [ ] **規格準拠のレイアウト**: ISO/IEC規格に基づく正確なハブ間隔（42.5mm）、キャプスタン位置、ネジ位置をピクセル単位で再計算・配置。
- [ ] **AXIAスタイルの外観**: 
    - 大きくクリアな窓（アシンメトリーなデザイン、またはAXIA特有のフレーム形状）。
    - 鮮やかな配色（蛍光色やメタリックな質感など、AXIAらしいアクティブな印象）。
    - スタイリッシュなラベルフォントとグラフィック。
- [ ] **物理的なディテールの深化**: 二重成形のシェルのような質感や、ハブの精密な造形（3本のスリットなど）。

## 技術的なアプローチ
1. 100.4mm(幅) = 480px を基準スケール（1mm = 4.78px）として全てのパーツ座標を再定義。
2. `index.html` の構造を刷新し、シェル（外殻）とインナーパーツ、ラベルを分離。
3. `css/style.css` でグラスモーフィズムを極限まで活用し、AXIA特有のクリアで透明感のある質感を表現。
4. `--accent-primary` 等の変数を AXIA らしい配色（サイバーな鮮やかさ）に微調整。

## 備考
- AXIAは80~90年代に若者向けに人気を博した富士フイルムのブランドであり、そのデザイン言語（スリムケース、洗練されたフォント、機能的な造形）を現代的に再釈釈する。
