# Issue: 単一トラック削除時の確認ダイアログ削除

## 概要
個別のトラックを削除する際、毎回確認ダイアログが表示されるのは操作の妨げになるため、これを削除する。

## 現状
- `deleteTrack` 関数内で `confirm()` が呼ばれており、削除のたびにOKをクリックする必要がある。

## 期待される動作
- トラックの削除ボタン（ホバー時に表示される×）をクリックすると、確認なしで即座にトラックが削除される。
- 一括削除機能については、引き続き確認ダイアログを表示する。

## 修正方針
- `js/app.js`: `deleteTrack` 関数の冒頭にある `confirm` チェックを削除する。
