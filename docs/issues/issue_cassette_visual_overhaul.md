## 概要
カセットテープの見た目をよりリアルでプレミアムなものへとブラッシュアップする。現在の簡易的な矩形と円の組み合わせから、細部（ネジ、テープの質感、リールの構造、窓の奥行きなど）にこだわったデザインへと進化させる。

## コンテキスト / 現状
現在のカセットモードは形状が単純で、本物のカセットテープが持つ「メカニカルな魅力」や「アナログ感」が不足している。ユーザーからも「再現度が低い」との指摘があり、改善が必要。

## 期待される動作 / ゴール
- [ ] カセット本体の造形：四隅のネジ穴、カセット下部のテープ露出部などのディテールを追加。
- [ ] テープリールの進化：ハブの形状（6本の歯があるタイプなど）や、巻き取られたテープの質感を表現。
- [ ] 窓とラベル：ラベルの質感を紙っぽくし、窓に奥行きや反射（グラスモーフィズム）を追加。
- [ ] アニメーション：再生時にリールが回転するだけでなく、テープの残り具合が動的に変化（またはよりらしく見える）ようにする。

## 技術的なアプローチ
1. `index.html` に必要な装飾要素（ネジ穴、ハブの詳細など）を追加。
2. `css/style.css` で擬似要素や高度なグラデーションを駆使してディテールを実装。
3. リールのハブ部分をSVGまたは複雑なCSSで描画し、よりメカニカルな印象を与える。

## 備考
- 再現度を高めるため、一般的な90分テープ（Maxell UDIIやTDK ADなど）の意匠を参考にモダンなアレンジを加える。
- α版運用に従い、実装後は即座に `main` へマージする。
