# Issue: ビジュアルテーマ追加 - 勉強机＆CD-R (Y2K)

## 概要
2000年代初頭の「勉強机でPCを開きながら、CD-Rのラベルを書き込んだり、プリンタからラベル印刷している」雰囲気を再現した新しいビジュアルテーマを追加します。

## 再現手順 / 現状
現在はデフォルトのガラスモーフィズム/サイバーパンクなデザインのみであり、ユーザーがより懐かしさを感じられる多様なシーン（ISSUE）を求めています。

## 期待される動作
- 勉強机、デスクトップPC、CD-R、プリンタなどの要素を盛り込んだUIデザイン。
- Y2Kエステティックに基づいた配色とタイポグラフィ（当時のOSのデザイン要素など）。
- CD-Rのラベルを書き込んでいるような手書き感のあるグラフィックやアニメーションの導入。
- 74minの既存機能（プレイリスト表示など）と調和する。

## 技術的な原因 / 詳細
特になし。新規デザインの追加。

## 修正方針 / 実装方針
1. 背景画像またはリッチなCSSでのシーン構成（勉強机の俯瞰視点など）。
2. CD-RやラベルプリンタをモチーフにしたUIコンポーネント（トラックリストをラベル風にする等）の設計。
3. `css/style.css` へのテーマ切り替え用クラスの追加、または新規CSSファイルの作成。
4. `js/app.js` でのテーマ選択UIの拡張。
