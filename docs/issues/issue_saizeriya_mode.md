# Issue: 「サイゼリヤ風」サクサク軽量モードの導入

## 背景
スペックの低い端末や通信環境が悪い場所でも快適に使用できるよう、リソース消費を最小限に抑えた動作モード（軽量モード）を導入します。

## 完了条件
- [x] 軽量モード切り替えスイッチの実装
- [x] アニメーション（CDの回転、モーダルのフェード等）を無効化するCSSクラスの導入
- [x] グラデーションやブラー（Glassmorphism）を単色や単純な透過に置き換えるテーマの作成
- [ ] 不要なDOM要素の非表示または簡略化

## 設計上の決定
- `body` に `.light-mode` などのクラスを付与し、CSS変数で全ての「重い」スタイルを上書きする。
- JS側でも重い計算やDOM操作をスキップするように条件分岐を追加する。
