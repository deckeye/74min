# Issue: スマートスピーカー（Google Home/Alexa）連携の検討 (#26)

## 概要
Google Home や Alexa などのスマートスピーカーから、74min で作成したプレイリストを再生できる機能の検討・実装。

## 背景
74min は音楽体験を重視したプラットフォームであり、リビングルームなどのスマートスピーカー環境での利用は親和性が高い。モバイルからのキャストや音声操作による再生をサポートすることで、日常的な利用頻度を高める。

## 期待される動作 / ゴール
- [ ] Google Cast (Chromecast built-in) 対応の検討
- [ ] Alexa Skill 開発の可能性調査
- [ ] スマホからの「キャスト」ボタンの実装
- [ ] 音声コマンドによるアルバム名指定での再生

## 検討事項
- YouTube IFrame API のキャスト制限の確認
- Web Receiver SDK の実装コスト
- Supabase API を経由したスマートスピーカーからのデータ同期

## 備考
- Media Session API との更なる連携強化が必要。
