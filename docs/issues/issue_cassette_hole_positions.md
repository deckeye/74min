## 概要
カセットテープの各パーツ（ハブ、ネジ、ポート部、および不足している穴など）の位置関係を正確な規格（IEC 60094等）に合わせて修正・追加します。

## コンテキスト / 現状
現在のカセットテープのUIは、リールのハブ間隔やネジの位置が目分量で配置されており、本物のカセットテープと比較すると違和感があります。特に「穴の位置がおかしい」というフィードバックを受けており、写実性を高めるためには正確なジオメトリの再現が必要です。

## 期待される動作 / ゴール
- [ ] ハブ（リール）の中心間距離を規格比率（幅の約41.6%）に合わせて修正。
- [ ] ネジ（4箇所）の位置を、カセットシェルの形状に合わせて適切に配置。
- [ ] キャプスタン穴、ガイドピン穴などの「不足している穴」の要素を追加し、適切な位置に配置。
- [ ] カセット下部の台形部分（ポート部）と本体の接続バランスを調整。

## 技術的なアプローチ
1. カセットテープの標準規格寸法（102mm x 64mm、ハブ間隔42.5mm等）に基づき、現在のpx単位のサイズ（幅480px）から各パーツの理想的な座標を計算する。
2. `index.html` にキャプスタン穴やピン穴を表現する `div` 要素を追加。
3. `css/style.css` で `position: absolute` と `top/left/transform` を使用してミリ単位の精度（に近い感覚）でパーツを再配置。
4. ホバー効果や照明の当たり方を微調整し、穴の「奥行き」を表現する。

## 備考
- 参考規格: IEC 60094-7
- デザイン指針の「Premium Feel」を維持しつつ、メカニカルな正確さを追求する。
