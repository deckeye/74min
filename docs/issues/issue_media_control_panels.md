## 概要
CD-Rモードとカセットモードにおいて、それぞれのメディアの下にプレイヤーを模したコントロールパネル（UIボタンと表示パネル）を配置し、直感的な操作を可能にする。

## コンテキスト / 現状
現在のUIでは、メディア（CD/カセット）が表示されているものの、再生・停止・曲送りなどの基本操作を行うための専用コントロールパネルが存在せず、プロジェクトの「かっこいいソフトウェア」「プレミアム感」をさらに高める余地がある。

## 期待される動作 / ゴール
- [x] CD-Rモード：CDラジカセ風のデジタル表示とコントロールボタン（Play/Pause, Stop, Prev, Next）がメディアの下に表示される。
- [x] カセットモード：レトロなラジカセ風の物理ボタン（Rew, Play, FF, Stop/Eject）がメディアの下に表示される。
- [x] 各ボタンが既存のYouTube再生ロジックと正しく連動する。
- [x] メディアの3D回転アニメーションに追従する。

## 技術的なアプローチ
1. `index.html` に各モード用のコントロールパネルHTML構造を追加。
2. `css/style.css` でプレミアム感のあるデザインを実装（グラデーション、シャドウ、ホバー時の視覚効果）。
3. `js/app.js` にイベントリスナーを追加し、既存の `togglePlay` や新設の `stopPlayback`, `playPrevTrack` 等に接続。
4. `switchMedia` 関数を更新し、モード切り替え時に適切なパネルを表示。
