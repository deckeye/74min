# Issue: ビジュアルテーマ追加 - ラジカセ＆カセットテープ (Retro 80/90s)

## 概要
80~90年代の「ラジカセのある部屋で、カセットテープに貼るラベルを書いたり録音したりしている」雰囲気を再現した新しいビジュアルテーマを追加します。

## 再現手順 / 現状
既存のデジタルなデザインに対し、よりアナログでレトロな質感を好むユーザー向けの選択肢を増やします。

## 期待される動作
- ラジカセ、カセットテープ、インデックスカードなどの要素を盛り込んだUIデザイン。
- 80~90年代のレトロな配色（パステルカラーや原色使い）、アナログ特有のノイズ感やテクスチャの導入。
- カセットテープのインデックスを書き込んでいるようなレトロなフォントや手書き風要素。
- テープが回転するようなマイクロアニメーションの導入。

## 技術的な原因 / 詳細
特になし。新規デザインの追加。

## 修正方針 / 実装方針
1. ラジカセやカセットテープを主役にしたシーンの構築。
2. インデックスカードを模したトラックリスト表示。
3. CSSフィルタを用いたアナログ感（グレインノイズなど）の演出。
4. 既存の再生ロジックと連動したカセットの回転アニメーションの実装。
