# YouTube Data API v3 設定ガイド

YouTube検索機能を有効にするためのAPIキー取得手順です。
**YouTube Premiumへの加入は不要**です。無料のGoogleアカウントがあれば誰でも取得できます。

## 手順

1. **Google Cloud Console にアクセス**
   - URL: [https://console.cloud.google.com/](https://console.cloud.google.com/)
   - Googleアカウントでログインしてください。

2. **プロジェクトの作成**
   - 画面左上のプロジェクト選択プルダウンをクリックし、「新しいプロジェクト」を選択。
   - プロジェクト名（例: `74min-dev`）を入力し、「作成」をクリック。

3. **YouTube Data API v3 の有効化**
   - 左側メニューの「APIとサービス」 > 「ライブラリ」を選択。
   - 検索バーに `YouTube Data API v3` と入力して検索。
   - 検索結果から選択し、「有効にする」をクリック。

4. **認証情報（APIキー）の作成**
   - API有効化後の画面、または「APIとサービス」 > 「認証情報」を選択。
   - 画面上部の「認証情報を作成」 > 「APIキー」を選択。
   - **`AIzaSy...` で始まるキーが表示されます。これがAPIキーです。**

5. **（推奨）APIキーの制限**
   - 作成されたキーの「キーを制限」をクリック。
   - **アプリケーションの制限**: 開発中は「なし」で構いませんが、公開時は「HTTPリファラー」でドメインを制限することを推奨します。
   - **APIの制限**: 「キーを制限」を選択し、`YouTube Data API v3` のみをチェックして保存。

## 次のステップ
取得したAPIキーを開発者に共有するか、設定ファイルに記述してください（今回のプロジェクトでは `js/config.js` を作成予定）。
